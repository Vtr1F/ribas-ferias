# Ribas Ferias

## Vaction Control System
### (Sistema de COntrolo de Férias)

- Sprint 1: Criação e Conexão da base de dados com o back-end, Conexão do back-end com o front-end e autenticação de usuários.

- Sprint 2: Roles de Usuário (Junto de Suas Limitações/Permissões), Criação de Novos Usuários, Edição de Informações no Perfil e Listagem de Usuários.

- Sprint 3: Criação de Pedidos de Férias/Ausências, Listagem e Gestão de Pedidos, Associação de Responsáveis a colaboradores, criação, listagem e gestão de Equipas.

- Sprint 4: Definição de Dias Máximos e Mínimos de Férias, Avisos sobre férias sobrepostas na mesma equipa, Definição do Calendário para dias bloqueados.

- Sprint 5: Testes finais, garantia de usabilidade, validação de funcionalidades e melhorias necessárias.
